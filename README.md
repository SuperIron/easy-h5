# easy-h5

## Description

基于 Vue 2.x + ts 搭建，可快速进行二次开发的 h5 框架，实现以下功能：

1. [x] 背景音乐

    - [x] 微信浏览器，自动播放

    - [x] 非微信浏览器，点击屏幕播放

2. [ ] 资源预加载

    - [ ] 图片预加载

    - [ ] 音频预加载

3. [ ] 微信 api

    - [ ] 登录

    - [x] 分享

4. [ ] 文件上传
